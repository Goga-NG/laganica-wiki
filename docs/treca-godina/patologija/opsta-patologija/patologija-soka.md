# Patologija šoka

ŠOK (SHOCK)
Stanje opšte hipoperfuzije organizma uzrokovano kardiovaskularnim kolapsom. Nemogućnost srca da održi minutni volumen (MV) i zadovolji metaboličke potrebe tkiva dovodi do prelaska na anaerobni metabolizam, nakupljanja laktata i laktatne acidoze.

TIPOVI ŠOKA
1. Hipovolemijski: gubitak krvi (trauma, krvarenje), gubitak plazme (opekotine), dehidracija, poliurija.
2. Kardiogeni: infarkt miokarda, ruptura srca, srčana insuficijencija.
3. Septički: infekcije (posebno Gram negativne bakterije).
4. Distributivni:
   - Anafilaktički (I tip preosetljivosti).
   - Neurogeni (gubitak vazomotornog tonusa: oštećenje kičmene moždine, anestezija).

STADIJUMI ŠOKA
1. Inicijalni (I faza): kompenzatorni mehanizmi aktivni, perfuzija vitalnih organa očuvana.
2. Progresivni (II faza): kompenzacija otkazuje, tkivna hipoperfuzija, metabolički poremećaji.
3. Ireverzibilni (III faza): teška ćelijska/tkivna oštećenja, fatalan ishod uprkos terapiji.

KLINIČKI ZNAKOVI (RANI STADIJUMI)
* Dominantni mehanizmi: kateholamini, RAAS, ADH.
* Simptomi: tahikardija, periferna vazokonstrikcija, hladna/bleda koža, oligurija.
* Izuzeci:
    - Septički i neurogeni šok: topla i crvena koža.
    - Kardiogeni šok: često izostaje tahikardija.
    - Anafilaktički šok: svrab, gušenje, gastrointestinalni simptomi (histamin).

PATOGENEZA PROGRESIVNOG ŠOKA
Hipoksija vitalnih organa → anaerobni metabolizam → laktatna acidoza → poremećaj vazomotora → vazodilatacija → staza krvi na periferiji → oštećenje endotela → DIK i MODS (Multiple Organ Dysfunction Syndrome).

MORFOLOGIJA
* Mozak: mikrovaskularna krvarenja.
* Srce: subendokardne hemoragijske nekroze.
* Bubrezi: akutna tubulska nekroza (akutna bubrežna insuficijencija).
* Pluća ("Šokna pluća"): kod sepse/traume; tvrda, kongestirana, hijaline membrane u alveolama.
* GIT: hemoragijska nekroza.
