# Termička oštećenja

TERMIČKA OŠTEĆENJA

I. POVIŠENA TEMPERATURA
Opšte dejstvo: TOPLOTNI UDAR
* Uzrok: Visoka temperatura okoline + visoka vlažnost + smanjeno odavanje toplote.
* Morfologija: Hipoksijsko oštećenje (nekroza ganglijskih ćelija), perivaskularna krvarenja (purpura cerebri), opšta vazodilatacija → hipovolemijski šok i DIK.

Lokalno dejstvo: OPEKOTINA (COMBUSTIO)
* Zone rane: Centralna (koagulaciona nekroza), zona inflamacije, zona hiperemije.
* Stepeni:
    1. I stepen (erythematosa): Crvenilo, edem.
    2. II stepen (bullosa):
       - IIa: Plikovi, adneksi očuvani.
       - IIb: Plikovi, adneksi oštećeni.
    3. III stepen (escharotica): Nekroza kože, tromboza, oštećenje nerava (keloidi pri zarastanju).
    4. IV stepen (carbonificatio): Ugljenisanje.
* Komplikacije: Hipovolemijski šok, sekundarne infekcije.

II. SNIŽENA TEMPERATURA
Mehanizam oštećenja:
* Direktno: Inhibicija enzima, kristalizacija vode (fizičko oštećenje ćelija).
* Indirektno (cirkulatorno): Vazokonstrikcija pa vazodilatacija (histaminoliberacija), povećan permeabilitet kapilara, endotelna nekroza → tromboza → infarkt.

Hipotermija (opšte dejstvo)
Definicija: Pad bazalne temperature (hipotalamus) ispod 35°C.
* < 35°C: Vazokonstrikcija, povećana termogeneza.
* < 33°C: Usporen metabolizam, pad pulsa, TA, disanja i diureze.
* < 25°C: Srčane aritmije, zastoj srca (cardiac arrest), depresija respiratornog centra.

Promrzline (lokalno dejstvo - Congelatio)
Najčešće na perifernim delovima tela (šake, stopala, nos, uši).
1. I stepen (erythematosa): Edem i hiperemija.
2. II stepen (bullosa): Nastanak plikova.
3. III stepen (gangrenosa): Nekroza kože i dubljih struktura.
