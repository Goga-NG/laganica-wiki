# Molekularne osnove i biohemijski mehanizmi genetskih bolesti

GENETSKE BOLESTI
Bolesti uzrokovane poremećajima naslednog materijala (jedro).

MUTACIJE (Nivo gena)
Permanentna oštećenja DNK; spontana ili indukovana (mutageni).
1. Missense (pogrešnog smisla): zamena baze → zamena aminokiseline u proteinu.
2. Nonsense (bez smisla): zamena baze → pojava stop kodona → prekid sinteze proteina.
3. Frame shift (pomeranje okvira): insercija/delecija baze → potpuno izmenjen "čitački okvir" → novi/nefunkcionalan protein.

ABERACIJE HROMOZOMA
1. Numeričke (odstupanje od 46 hromozoma):
   - Monozomija: nedostatak hromozoma (npr. Tarnerov sindrom 45,X).
   - Trizomija: prekobrojni hromozom (npr. Daunov 21, Edvardsov 18, Patau 13).
   - Klinefelterov sindrom (47,XXY): dizomija X hromozoma kod muškaraca.
2. Strukturne:
   - Translokacije (razmena delova između različitih hromozoma).
   - Inverzije (premeštanje unutar istog hromozoma).
   - Delecije (gubitak dela hromozoma).
   - Izohromozomi (samo kratki ili samo dugi kraci usled poprečne podele).

BIOHEMIJSKI MEHANIZMI
1. Enzimopatije (Enzimski defekti):
   - Smanjena aktivnost enzima dovodi do nakupljanja toksičnih intermedijera metabolizma (npr. glikogenoze, lizozomske bolesti nakupljanja).
2. Defekti membranskih receptora i transporta:
   - Smanjena sinteza/funkcija receptora ili transportnih proteina.
   - Primer: mutacija hloridnog kanala kod cistične fibroze.
3. Defekti neenzimskih proteina:
   - Strukturni proteini: npr. fibrilin kod Marfanovog sindroma.
   - Funkcionalni proteini: npr. hemoglobin kod anemije srpastih ćelija.
